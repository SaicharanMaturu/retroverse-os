### 2. 🎯 **Advanced Terminal Commands**

**File Search:**
- `grep pattern filename` - Search for text patterns in files
- `find pattern` - Search for files by name

**File Operations:**
- `head filename [lines]` - Show first N lines (default 5)
- `tail filename [lines]` - Show last N lines (default 5)
- `wc filename` - Count lines, words, characters
- `sort filename` - Sort file contents
- `uniq filename` - Show unique lines only
- `chmod permissions filename` - Change file permissions (777, 755, etc.)

**Example Usage:**
```
> grep "hello" myfile.txt
> find *.txt
> head script.py 10
> tail logfile.txt 20
> wc project.json
> chmod 755 script.sh
```
